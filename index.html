CSS (Cascading Style Sheets) is used to style and layout web page. it is used to alter the font, size, color and spacing of your content, and to add other decorative features.

Paddings

Paddings are used to generate space around an element's content, inside of any defined borders. there are four properties for seting padding for each side of an element: padding top, padding right, padding bottom and padding left.

Margins

The margin properties are used to create space around elements, outside of any defined borders. The margin top, margin right, margin bottom and margin left are used for setting the margin for each side of an element.

Borders

The CSS borders helps to specify the style, width and color of an element's border.
